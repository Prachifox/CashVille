# CashVille
An online shopping website . <br/>
It has both customer and admin end.<br/>

#### Customer can :
View products <br/>
purchase products <br/>
provide feedback <br/>
reset their password <br/>

#### Admin can :
Add products <br/>
Update product information <br/>
View products <br/>
Delete products <br/>
Reset password. <br/>

#### Technologies used: 
##### Frontend:
Servlet <br/> 
JSP <br/>
Materialize Framework. <br/>

##### Backend:
Spring MVC Framework <br/>
Maven <br/>
Javamail <br/>
Hibernate with H2 database <br/> 

##### Server:
Apache Tomcat. <br/>
